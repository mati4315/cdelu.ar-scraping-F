Comando	Uso	
npm start		Producción: scrapea y guarda en MySQL
npm run dry		Prueba: mismo flujo pero sin tocar la DB
npm run debug		Producción con logs detallados (nivel debug)
npm run dry:debug	Prueba con logs detallados, sin tocar DB
npm run debug:init	Crea/recrea las tablas en MySQL si no existen
